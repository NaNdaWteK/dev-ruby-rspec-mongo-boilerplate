# Start Ruby Development project

That is the initial configuration for a tested ruby app with rspec and mongo database

## Up the app
  `$ make up`

#### Run all test
  `$ make test`

#### Run tdd test
  `$ make tdd`

#### Run integration test
  `$ make integration`

#### Run feature test
  `$ make feature`

#### Run a interactive shell
  `$ make shell`
  
#### Visit app
  `localhost:4567`

## Down the app
  `$ make down`
